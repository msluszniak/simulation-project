# simulation-project

Co wymaga dopowiedzenia to to jak trackować animala. Aby trackować animala i żeby po określonej liczbie epok wypisało na konsole jego statystyki należy najpierw kliknąć
w pole z tekstem (nawet jeśli jest wpisany default np. 100). Następnie kliknąć w wybranego animala i kliknąć button "Wartość n". Czasem za pierwszym razem nie załapuje mi tackowanego
animala od razu, więc należy spróbować ponownie.
